Разработать:
 
    - Структуру базы данных для хранения данных об авторах и книгах.
      Каждый автор может написать несколько книг и у каждой книги может быть несколько авторов.
      
    - Приложение на ASP.Net Core, которое предоставляет REST API:
      а. для добавления, редактирования и удаления данных из БД;
      б. для отображения количества книг определенного автора;
    
    - В качестве СУБД желательно использовать PostgreSQL.
